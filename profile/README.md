FunDive is a Biodiversa+ funded project aiming to establish integrative platform to investigate how fungi can be included in monitoring and biodiversity conservation across Europe. Here we will list repositories related to different work packages of the project.

Homepage: https://fun-dive.eu

Page at Biodiversa+: https://www.biodiversa.eu/2024/04/15/fundive
